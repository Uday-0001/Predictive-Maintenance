Predictive-Maintenance


This Streamlit application provides an AI-powered predictive maintenance system designed to monitor machine health and predict potential failures before they occur. The dashboard offers real-time monitoring, failure risk predictions, performance analytics, and batch processing capabilities.


Model Performance
The dashboard displays a model accuracy of 94.2% with an AUC score of 0.87, indicating strong predictive capability. The ROC curve visualization in the Analytics tab demonstrates the model's ability to distinguish between failure and non-failure states across various threshold settings.
